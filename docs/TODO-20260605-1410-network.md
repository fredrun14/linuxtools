### PR1 — ROBUSTESSE (🟠 MAJEUR)

#### `repository.py` — écriture atomique (l.74)
```python
import os
import tempfile

def save(self, devices: list[NetworkDevice]) -> None:
    """Écrit l'inventaire de façon atomique (tmp + os.replace)."""
    data = json.dumps([d.to_dict() for d in devices], ensure_ascii=False)
    fd, tmp = tempfile.mkstemp(dir=str(self._path.parent), suffix=".tmp")
    try:
        with os.fdopen(fd, "w", encoding="utf-8") as f:
            f.write(data)
        os.chmod(tmp, 0o644)
        os.replace(tmp, self._path)  # atomique
    except BaseException:
        os.unlink(tmp)
        raise
```

| Cas | Condition | Action |
|---|---|---|
